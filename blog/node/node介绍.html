# 初始化项目

npm init
或
npm init -y
1
2
3
  • 初始化后生成package.json文件
点击查看package.json的作用
  1. 记录项目的一些信息，安装了哪些包等
  2. 项目传递时不需要发送依赖包，接收到项目后执行npm install就可以将项目依赖全部下载到项目里
  3. 初始化后npm会把依赖安装到此文件夹内
