# node->install

# 1.下载安装包

点击进入官网 (opens new window)，然后傻瓜式安装。

# 2.配置npm在安装全局模块时的路径和缓存cache的路径

在node.js安装目录下新建两个文件夹 node_global和node_cache，打开cmd并执行
npm config set prefix "D:\node.js\addr\node_global"
npm config set cache "D:\node.js\addr\node_cache"
系统变量中新建变量名为 “NODE_PATH”， 值为“D:\node.js\addr\node_modules”，不需要
用户变量里的Path，将相应npm的路径为：D:\node.js\addr\node_global
用户变量里的Path，将相应npm的路径为：D:\node.js\addr\node_cache

在cmd命令下执行 npm install webpack -g 然后安装成功后可以看到自定义的两个文件夹已生效。 执行 npm webpack -v 可以看到所安装webpack的版本号
